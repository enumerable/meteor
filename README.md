# Meteor 非官方中文文档项目

Meteor 是个极其简单的、用来搭建现代 Web 应用的环境。

在 Meteor 的帮助下，你的应用将：

* 用纯 Javascript 写成
* 在网络上只传输必要数据，而非 HTML
* 随意搭配你喜爱的其他开源库

官方文档可于此处访问：[https://docs.meteor.com/](https://docs.meteor.com/)

非官方中文文档可于此处访问：[https://d0cs.meteor.com/](https://d0cs.meteor.com/)
